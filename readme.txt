============================================================================================
ABOUT
============================================================================================

This pack provides voxel replacements for sprites in Shadow Warrior.
It is meant to be used with the VoidSW port.


============================================================================================
HOW TO USE
============================================================================================

VoidSW:
- put voxels.zip into autoload folder
- make sure "enable autoload folder" checkbox is enabled
